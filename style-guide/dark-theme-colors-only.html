Este guia contém apenas mapeamento de cores. Não altere estrutura HTML, componentes, espaçamentos, border-radius, sombras ou qualquer outro elemento visual além das cores listadas aqui.

🔄 Mapeamento Light → Dark
✅ Como Usar
Substitua as cores do seu tema light (coluna "De") pelas cores dark correspondentes (coluna "Para"). Mantenha toda a estrutura HTML/CSS existente.
Elemento De (Light) Para (Dark) Uso
Body Background #FFFFFF ou #F4F5F7 → #0A0A0A Fundo principal da página
Card Background #FFFFFF → #18181B Backgrounds de cards, painéis
Sidebar Background #FFFFFF ou #F9FAFB → #18181B Background de sidebars
Modal/Dialog BG #FFFFFF → #1C1C1E Modais, dropdowns, tooltips
Hover Background #F3F4F6 ou #F9FAFB → #27272A Hover em listas, menu items
Active Background #E5E7EB → #3F3F46 Item selecionado, active state
Border Padrão #E5E7EB ou #D1D5DB → #27272A Bordas de elementos
Border Hover #D1D5DB ou #9CA3AF → #3F3F46 Bordas em hover/focus
Divisor/Separator #F3F4F6 ou #E5E7EB → #1F1F23 ou #27272A Linhas divisórias
Texto Primário #111827 ou #000000 → #FAFAFA Títulos, headings
Texto Secundário #374151 ou #1F2937 → #E5E7EB Corpo de texto
Texto Terciário #6B7280 → #A1A1AA Metadados, labels
Texto Muted #9CA3AF → #71717A Texto desabilitado
Placeholder #9CA3AF ou #D1D5DB → #52525B Placeholders de input
Primária (Brand) #3B82F6 → #3B82F6 Mantém a mesma (já funciona)
Primária Hover #2563EB → #2563EB Mantém a mesma
Verde Sucesso #22C55E ou #10B981 → #10B981 Mensagens de sucesso
Amarelo Warning #F59E0B → #F59E0B Mantém a mesma
Vermelho Erro #EF4444 → #EF4444 Mantém a mesma
💻 CSS Variables (Recomendado)
✅ Melhor Prática
Use CSS variables para facilitar a manutenção. Crie estas variáveis no :root e substitua todas as cores hard-coded por variáveis.
/* ===== DARK THEME - CSS VARIABLES ===== */ :root { /* Backgrounds */ --bg-primary: #0A0A0A; --bg-elevated: #18181B; --bg-elevated-2: #1C1C1E; --bg-hover: #27272A; --bg-active: #3F3F46; /* Borders */ --border-subtle: #1F1F23; --border-default: #27272A; --border-hover: #3F3F46; --border-active: #52525B; /* Text */ --text-primary: #FAFAFA; --text-secondary: #E5E7EB; --text-tertiary: #D4D4D8; --text-muted: #A1A1AA; --text-subtle: #71717A; --text-placeholder: #52525B; /* Brand Primary (Azul) */ --primary: #3B82F6; --primary-hover: #2563EB; --primary-pressed: #1D4ED8; --primary-bg-subtle: rgba(59, 130, 246, 0.1); /* Semantic Colors */ --success: #10B981; --success-bg: rgba(16, 185, 129, 0.1); --warning: #F59E0B; --warning-bg: rgba(245, 158, 11, 0.1); --error: #EF4444; --error-bg: rgba(239, 68, 68, 0.1); } /* Exemplo de uso */ body { background: var(--bg-primary); color: var(--text-secondary); } .card { background: var(--bg-elevated); border: 1px solid var(--border-default); } .button-primary { background: var(--primary); color: var(--text-primary); } .button-primary:hover { background: var(--primary-hover); }
